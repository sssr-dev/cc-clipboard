### Run
```shell
$ python3 main.py start  # start program

$ python3 main.py stop   # stop program

```

### Install

```shell
# Shell

$ git clone https://github.com/sssr-dev/cc-clipboard.git
  # Cloning repo
$ cd src
$ pip3 install -r requirements.txt
  # Install all dependents
```
